Verifying a Bitcoin address

*Bitcoin addresses can be up to 34 Characters.
*Bitcoin addresses are most often 33 or 34 characters long with 34 being the max length. (Source: Bitcoin.it/wiki/List of address prefixes)
Pseudo Code
input_address = 1PZoWjye1Xa1e1NSNYmKuxEJYGvcnJro7p;
address_valid = true;

if input_address contains the following characters 0 O I l
address_valid = false;

if input_address string length is greater than 34
address_valid = false;

if input_address first character is not a number 1
address_valid = false;

return address_valid



Create a Bitcoin address

Generalized steps involved to generate a Bitcoin address

1] Generate a Private key.
*By generating a Private key, you recieve both a Private and a Public key.

2] Perform a SHA-256() Hashing operation on the public key, We will call this value SHA256-PublicKey

3] Perform a RIPEMD-160() Hashing operation on the SHA256-PublicKey, We will call this value RIPEMD160-PublicKey

4] Add 00 to the front of the RIPEMD160-PublicKey (Notes: We are describing generating bitcoin addresses, to generate a testnet address add a 6f instead and for namecoin add a 34 instead)

5] Perform SHA-256() Hashing operation on the RIPEMD160-PublicKey (with the 00 included)
